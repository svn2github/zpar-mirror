1 The Basics of Zpar

ZPar is a statistical natural language parser, which performs syntactic analysis tasks including word segmentation, part-of-speech tagging and parsing. ZPar supports multiple languages and multiple grammar formalisms. ZPar currently supports context free grammar (CFG), dependency grammar and combinatory categorial grammar (CCG).

2 System Requirements

The ZPar software requires the following basic system configuration

  • Linux/GCC
  • 256MB of RAM minimum
  • At least 500MB of hard disk space

3 Download and Installation

Download the latest tar ball from sourceforge and move the tar ball to your work space.
You can use ZPar by referring to quick start, or follow detailed instructions in individual modules for installation and usage.

  • Chinese word segmentation
  • POS tagging
  • dependency parsing
  • phrase-structure parsing
